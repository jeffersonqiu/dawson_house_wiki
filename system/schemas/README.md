# Schemas

Data contracts for extracted facts. YAML for human review; parseable when the harness is built.

| Schema | Use |
|--------|-----|
| [item.yaml](item.yaml) | Room item (appliance, fitting, furniture) |
| [vendor.yaml](vendor.yaml) | Supplier or contractor |
| [task.yaml](task.yaml) | Follow-up or action item |

Extracted records should validate against these before compilation.
