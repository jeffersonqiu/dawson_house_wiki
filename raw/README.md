# Raw imports

Drop exported snapshots here. Do not organize renovation knowledge manually in this folder.

| Folder | Drop |
|--------|------|
| `sheets/` | CSV or JSON exports from Google Sheets |
| `drive/` | Markdown or text exports from Google Docs |

**Naming:** `{source}-{date}.{ext}` — e.g. `budget-2026-06-07.csv`

**Preferred:** MCP pull via Ingestion agent — see `MCP_SETUP.md`.

**Fallback:** export and copy files by hand.
